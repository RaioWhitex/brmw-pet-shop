# 📑 PDF ER Diagram Viewer

Um projeto de **Visualizador de Diagramas e Documentação PDF**, desenvolvido para facilitar a consulta de modelos conceituais, diagramas entidade-relacionamento (DER) e documentos técnicos diretamente no navegador.

Ideal para estudantes, analistas de sistemas, desenvolvedores e equipes que trabalham com modelagem de banco de dados e documentação de projetos.

---

## 📸 Preview do Projeto

Interface contendo:

- Visualização integrada de arquivos PDF
- Exibição de diagramas ER e modelos conceituais
- Barra de ferramentas completa
- Controle de zoom e navegação
- Opções de download e impressão
- Área de leitura responsiva

---

## ✨ Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **PDF.js**

---

## 🎨 Características do Design

- Interface moderna e profissional

- Compatibilidade com múltiplos documentos
- Experiência otimizada para leitura técnica

---

## 🚀 Objetivo do Projeto

Desenvolver uma solução simples e eficiente para exibição de documentos PDF, permitindo a análise de diagramas, modelos conceituais e documentação técnica sem a necessidade de softwares externos.

---

## 📚 Funcionalidades

- Carregamento de documentos PDF
- Navegação entre páginas
- Controle de zoom
- Download do arquivo
- Impressão direta
- Rolagem horizontal e vertical
- Suporte a diagramas de banco de dados
- Interface responsiva
